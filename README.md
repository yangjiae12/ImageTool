# ImageTool

MFC 기반 영상 처리 실습 프로젝트

<br>

## 3주차

#### BMP 파일 구조 이해
- `BITMAPFILEHEADER`, `BITMAPINFOHEADER`, 픽셀 데이터 구성 파악
#### BMP 파일 출력
- `SetDIBitsToDevice()`와 `StretchDIBits()`를 이용한 이미지 출력
#### 비트맵 처리 클래스 작성
- `IppDib` 클래스 구현 (비트맵 생성, 파일 입출력, 화면 출력 기능 포함)
#### 영상 출력
- `OnLButtonDown()` 함수에서 BMP 파일을 읽어 화면에 출력
